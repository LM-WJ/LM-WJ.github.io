搭建博客遇到的问题和解决办法

在这篇文章我会记录我搭建博客并美化时遇到的问题，以及解决方法

npm安装没反应

解决办法：更换淘宝源

hexo g 报错

can not read a block mapping entry; a multiline key may not be an implicit key

解决办法：在设置属性后面加空格

hexo d报错

Error: Spawn failed at ChildProcess

解决办法：

  • 删除文件夹 .deploy_git
  • 执行命令 git config –global core.autocrlf false
  • 然后就是 hexo clean
  • hexo g
  • hexo s
  • 最后查看没问题就可以部署到github上了
  • hexo d

博客无法访问

上周我为了让博客访问速度加快一点，于是就把博客项目从GitHub迁到了vercel，以及使用cloud flare来提供CDN加速。部署完之后就等着一天之后自动更新dns，结果第二天cloud flare和vercel陆续给我发来了邮件说域名配置出错。今天看了cloud flare的提示，将指向vercel的dns服务器删去，马上恢复了正常访问🤣。
